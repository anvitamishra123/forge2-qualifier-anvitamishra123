\# Architecture



\## Application



This project is a tiny Trello-style Kanban board built for Forge 2 qualifier.



\## Backend



\- Laravel REST API

\- SQLite database

\- APIs for boards, lists, cards, tags, members, due dates and card movement



\## Frontend



\- React Vite UI

\- Connects with Laravel API

\- Supports board/list/card flow, tags, member assignment, due dates and overdue flag



\## Agent Setup



Forge 2 uses two agents:



\- Hermes as the brain/orchestrator

\- OpenClaw as the hands/coding agent



\## Slack Channels



\- #sprint-main: human talks to Hermes

\- #agent-coder: OpenClaw coding tasks

\- #agent-log: logs, status and autonomous run evidence
